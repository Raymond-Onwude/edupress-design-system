# EduPress Design Language

EduPress is an education-first interface system for pupils, teachers,
parents and school administrators.

## Principles

### Warm, not childish
Interfaces should feel welcoming to young learners without resembling toys.

### Structured, not rigid
Every screen should clearly show purpose, sequence and progress.

### Calm motion
Animation should explain state changes and reward progress without distracting
from learning.

### Pedagogy before decoration
Visual hierarchy must reinforce the learning objective, activity and feedback.

### Accessible by default
Keyboard access, visible focus, readable contrast, clear labels and reduced
motion support are mandatory.

## Visual hierarchy

1. Cream application canvas
2. White or forest content panels
3. Semantic accent colours
4. Rounded educational controls
5. Short, purposeful motion

## Typography

- Fraunces: major lesson and document titles
- Baloo 2: headings, controls and friendly labels
- Andika: pupil-facing and long-form reading
- IBM Plex Mono: metadata, curriculum references and progress labels
