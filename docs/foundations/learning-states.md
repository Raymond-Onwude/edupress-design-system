# Learning States

EduPress colours communicate educational meaning.

| State | Primary token | Intended use |
|---|---|---|
| Discover | Blue | New concepts, listening and exploration |
| Practise | Coral | Pupil interaction and guided practice |
| Reflect | Gold | Thinking prompts and review |
| Progress | Green | Completion, growth and success |
| Teacher | Forest | Teaching plans and professional tools |
| Assessment | Forest | Rubrics, observation and formal checks |
| Canvas | Cream | Default reading and learning background |
| Error | Reserved | Incorrect or blocked states only |

Colour must never be the only way a state is communicated. Pair it with text,
an icon or another visible indicator.
