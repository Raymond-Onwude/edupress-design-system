# EduPress Design System

An education-first design system powering EduPress learning applications.

## Purpose

This repository provides shared:

- design tokens
- React UI primitives
- educational components
- learning-state patterns
- themes
- motion
- icons
- accessibility standards
- documentation

## Core packages

- `@edupress/tokens`
- `@edupress/ui`
- `@edupress/education`
- `@edupress/motion`
- `@edupress/icons`
- `@edupress/themes`

## Design foundations

The initial visual system is derived from the Primary 1 English learning
experience developed for Great Learners Academy.
